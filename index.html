Mis Productos

Tarta de Banana Split

Base de vainilla con una generosa capa de dulce de leche, acompañada de rodajas de banana en el centro, cubierta con suave crema chantilly, crocante de maní y adornada con guindas para un toque final irresistible.

Tarta Oreo

Deléitate con una base ligeramente cubierta de dulce de leche, coronada con una suave capa de crema chantilly mezclada con trozos de galletas Oreo trituradas y decorada con galletas Oreo enteras para un toque irresistible.

Tarta Ganash

Disfruta de una base de vainilla cubierta con una capa de dulce de leche, coronada con un ganash de chocolate semiamargo y decorada con rodajas frescas de frutilla para un toque final exquisito.

Tarta de Durazno

Una deliciosa base de vainilla con una capa de dulce de leche, cubierta con suave crema chantilly y decorada con rodajas de durazno fresco para un acabado irresistible.

Tarta Ganash

Disfruta de una base de vainilla cubierta con una capa de dulce de leche, coronada con un ganache de chocolate blanco semiamargo y decorada con delicados hilos de chocolate con leche para un toque final exquisito.

Tarta Ganash

Una base de vainilla con una capa de dulce de leche, cubierta con una espesa capa de ganash de chocolate negro semiamargo y decorada con suave crema chantilly para un toque especial.

Tarta Mix Durazno-Oreo

Disfruta de una base de vainilla con una capa de dulce de leche, coronada con crema chantilly mezclada con trozos de galletas Oreo trituradas y decorada con rodajas de durazno y galletas Oreo enteras para un acabado irresistible.

Tarta de Coco

Una deliciosa base de vainilla cubierta con una generosa capa de dulce de leche y finalizada con una capa de coco rallado al huevo para un sabor inigualable.

Rosca de Pascua/Reyes Magos

Una deliciosa base de vainilla, con opción de relleno de dulce de membrillo o sin relleno, cubierta con suave crema pastelera, trozos de membrillo, cerezas, coco rallado y delicados hilos de chocolate.

Eliana Díaz

Hola, soy Eliana Díaz, tengo 20 años y soy estudiante del profesorado de Geografía. Desde 2018, comencé este pequeño emprendimiento de repostería inspirado en mi amor por la dulzura desde niña. Me especializo en preparar tartas y roscas de Pascua, siempre con el objetivo de endulzar momentos únicos y especiales. A lo largo de estos años, he tenido el honor de ser parte de muchas celebraciones importantes: cumpleaños, fiestas de egreso e incluso bautismos. Cada pedido representa una oportunidad para acompañar a mis queridos clientes en sus festejos, aportando un toque dulce a sus momentos más memorables. Gracias por permitirme ser parte de sus historias.
